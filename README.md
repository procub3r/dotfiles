# dotfiles
All my configuration files - the so called "dotfiles" - needed to replicate my Linux desktop setup go here. This repo serves both as a backup for me, and also as a means to share my configs with the world!

## My setup uses
- BSPWM (Window Manager)
- Polybar (Status bar)
- Alacritty (Terminal Emulator)
- Neovim (Text Editor)
- Picom (Compositor)
